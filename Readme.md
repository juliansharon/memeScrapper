# Image Url Scrapper

## Domain

> https://image-url-scrapper.herokuapp.com/

## Endpoints

### urls

> /urls

This will return an image urls based on the search query

#### Example

##### Url

> POST https://image-url-scrapper.herokuapp.com/urls

##### Body

    {
        "query":"dogs"
    }

This will return the image urls of dogs.
